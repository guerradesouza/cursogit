# cursogit
cursogit pelo site

1 - Criar novo repositório no github
2 - git clone linkDoClone.git
3 - Entrar no repositório do projeto:
4 - git init
5 - git add *
6 - git commit -m "Primeiro Commit"
7 - git push

Obs: Cada vez que quiser atualizar tem que dar o:
git add *
git commit -m "Nome do commit"
git push
